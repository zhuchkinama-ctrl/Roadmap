# Требования

## Бизнес‑требование
- Приложение должно поддерживать аутентификацию пользователей с помощью JWT.

## Техническое требование
- Для генерации и валидации JWT необходимо использовать библиотеку **io.jsonwebtoken** (JJWT).
- При сборке проекта (`mvn clean install`) должна успешно проходить без ошибок компиляции, связанных с отсутствием пакета `io.jsonwebtoken`.

## Ограничения
- Использовать последнюю стабильную версию JJWT, совместимую с Java 17 и Spring Boot 3.
- Добавление зависимости должно быть оформлено в `pom.xml` проекта.
